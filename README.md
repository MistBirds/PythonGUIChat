# 本项目采用的语言以及框架：Python3, Tkinter
# 本项目拟实现:
### 1.用户的注册登陆
### 2.查找并添加好友
### 3.显示好友列表并获得在线状态
### 4.实现实时聊天

### 有一个基础的基于win命令行的版本
chat_client_cmd.py是基于win命令行的
### 还有一个复杂一些的gui版本
